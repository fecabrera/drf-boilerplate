# DRF Boilerplate

## Quickstart
```bash
# Cloning the repository
git clone https://github.com/fecabrera/drf-boilerplate.git

# Installing dependencies
cd drf-boilerplate
pipenv install --dev

# Preparing the environment
cp .env.example .env
docker-compose up -d

# Running the server
python manage.py runserver
```
